# Pong
## Revamp of the class game Pong with HTML Canvas/JS
## Version: 1.0
### Features:
* Completed
	* Random ball color
	* Random ball velocity
	* Ball/Paddle Collision
	* Basic Bot 
	* Scores 
	* Paddle move animation
	* Paddle hit animation
	* Backspin
	* Improved Score Board

* Todo:
